# Manual de Instalación y Personalización de Arch Linux desde VMware

Una guía técnica integral que documenta el proceso completo de instalación, configuración y personalización avanzada de Arch Linux utilizando BSPWM, Kitty, NvChad, Zsh y herramientas modernas de gestión de paquetes como Pacman, Yay y Paru.

---

## Descripción General

Este repositorio funciona como una base de conocimiento personal y profesional que documenta un flujo de trabajo reproducible para instalar y configurar Arch Linux desde cero. El contenido abarca los siguientes temas:

- Configuración inicial de la red Wi-Fi mediante `iwctl`
- Instalación automatizada utilizando `archinstall`
- Configuración del gestor de inicio de sesión `LightDM`
- Personalización visual del sistema mediante `RiceInstaller`
- Configuración avanzada del emulador de terminal `Kitty`
- Integración del entorno de desarrollo `NvChad`
- Personalización de la shell `Zsh`
- Sincronización del entorno del usuario `root`
- Gestión de paquetes con `Pacman`, `Yay` y `Paru`
- Herramientas opcionales para virtualización y proximamente pentesting

El proyecto está escrito en Markdown y es mantenido en Obsidian, lo que permite que la documentación funcione como una wiki técnica estructurada y como una referencia de largo plazo.

---

## Objetivos

- Crear un entorno de Arch Linux completamente reproducible.
- Documentar cada paso de configuración de forma clara, estructurada y profesional.
- Construir una base de conocimiento técnico personal.
- Desarrollar un proyecto de calidad de portafolio que demuestre habilidades en Linux y administración de sistemas.
- Proporcionar una base sólida para futuros estudios en ciberseguridad, ingeniería de software e infraestructura.
